## Installation

### Clone
- Clone this repository to your local machine or download the zip file 
> Windows
```bash
git clone https://github.com/Devansh3712/PyCrypt.git
```
> Linux
```shell
git clone https://github.com/Devansh3712/PyCrypt.git
```

### Setup
- In order to run the program, the modules mentioned in requirements.txt must be there. In order to install these modules, follow the steps:
> Windows
```bash
python -m pip install -r requirements.txt
```
> Linux
```shell
python3 -m pip3 install -r requirements.txt
```

![Setup GIF](https://github.com/Devansh3712/PyCrypt/blob/master/res/i.gif)

---

# Usage
- Now, you can either run the pycrypt.py file
> Windows
```bash
python pycrypt.py
```
> Linux
```shell
python3 pycrypt.py
```

- Or you can either run the `pycrypt.exe` file in ./dist/ or install it using PyScrypt installer, available as a <a href="https://github.com/Devansh3712/PyCrypt/releases/tag/v1.0" target="_blank"></a>.
